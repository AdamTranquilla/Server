# Tablewise

A mobile restaurant menu that allows users to place and split orders based on their seat numbers. Orders can be fully customized, and menus can be filtered based on dietary restrictions.
## Stack

### Frontend:
- React Native
- TypeScript
- HTML5
- Sass

### Backend:
- GraphQL
- Node
- Express 
- MongoDB
- Socket.io

## Testing:
- Jest
- Cypress
- Storybook
